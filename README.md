# Контролер-ТС — чат-бот для мессенджера MAX

Бот для оформления результатов осмотра оборудования тепловых сетей (ТК).

## Требования

- Go 1.21+
- CGO (для SQLite)

## Запуск

```bash
cd kontroler-ts
go mod tidy
go run main.go
```

Переменные окружения в `.env`:

- `TOKEN` — токен бота MAX
- `DB_PATH` — путь к SQLite (по умолчанию `./kontroler.db`)
- `UPLOADS_DIR` — каталог для загрузок (по умолчанию `./uploads`)

## Первый администратор

Добавьте телефон в `allowed_phones` и создайте пользователя с `role='admin'` в БД, либо загрузите Excel через меню администратора.

## Команды

- `/start` — сброс сессии и приветствие
- `/menu` — главное меню (для авторизованных)
